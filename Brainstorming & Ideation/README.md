# Brainstorming & Ideation

This folder contains the brainstorming and ideation artifacts created during
the initial phase of the MedSafe AI project.

These documents capture the design thinking process used to understand
the problem space, explore ideas, and prioritize the final project concept.

## Files Included

1. BrainstormingTemplate.pdf  
   Initial brainstorming session for identifying potential AI healthcare ideas.

2. Define Problem Statements Template.pdf  
   Defines the user-centered problem statements for the project.

3. Empathy Map Canvas.pdf  
   Captures the user's perspective, needs, and challenges through empathy mapping.

These artifacts represent the early ideation stage before moving into
project planning and system design.

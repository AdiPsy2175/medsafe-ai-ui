# MedSafe AI – Medical Safety Assistant (UI Module)

MedSafe AI is an educational, AI-powered medical safety assistant designed
to improve awareness about medicine interactions and symptom-based risk
guidance through an interactive web interface.

This repository contains the **Streamlit-based User Interface** developed
as part of the *Generative AI with Google Cloud* group project.

---

## 🚀 Live Demo
🔗 https://medsafe-ai-ui.streamlit.app

---

## 🎯 Project Objectives
- Provide a simple interface for checking possible medicine interactions
- Offer educational symptom and side-effect guidance
- Promote medical safety awareness without replacing professional advice

---

## 🧩 Features Implemented
### 💊 Medicine Interaction Checker
- Accepts multiple medicine names
- Displays interaction warning and risk level
- Handles empty input gracefully

### 🩺 Symptom & Side-Effect Guidance
- Accepts symptom descriptions
- Provides educational guidance
- Displays urgency level

# Milestone 1: Environment Setup

This document describes the environment setup and tools required
to develop and run the MedSafe AI project.

---

## Development Environment
- Operating System: Windows / Linux / macOS
- Programming Language: Python 3.9+
- Code Editor: VS Code (recommended)
- Version Control: Git & GitHub

---

## Required Tools & Libraries
- Streamlit (for user interface)
- Python standard libraries

All dependencies are listed in the `requirements.txt` file.

---

## Installation Steps
1. Install Python from https://www.python.org
2. Clone the repository:
git clone https://github.com/AdiPsy2175/medsafe-ai-ui
3. Navigate to the project directory:
   cd medsafe-ai-ui
5. Install dependencies:
   pip install -r requirements.txt

---

## Running the Application Locally
```bash
To run the application locally:
streamlit run app.py
```

## Deployment Environment

The application is deployed using **Streamlit Community Cloud**, which
automatically installs dependencies and hosts the application directly
from the connected GitHub repository.
